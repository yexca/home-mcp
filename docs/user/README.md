# User Guide

This guide is for running `home_mcp_gateway` as a local MCP gateway. Normal users configure the gateway through `.env`; the `config/` directory is kept for application-owned baseline files, ComfyUI workflow examples, and generated agent fragments.

## Files You Touch

| File | Purpose |
| --- | --- |
| `.env.example` | Committed template listing common user settings. Copy it to `.env`. |
| `.env` | Your local runtime settings, tokens, provider URLs, and module switches. Ignored by git. |
| `.env.agent.<name>` | Per-agent bearer and Matrix token file generated by `apply_agent.bat`. Ignored by git. |

## Files Managed By The Project

| File or directory | Purpose |
| --- | --- |
| `config/config.main.yaml` | Application baseline structure and defaults. It is loaded automatically. |
| `config/comfyui/` | ComfyUI workflow JSON examples used by local image generation. |
| `config/agent/` | Per-agent structured YAML generated by `apply_agent.bat`. |
| `tests/config/` | Test-only YAML fixtures used by the test runner. |

Do not create or edit `config/config.yaml` for normal use. Older versions used that file as the user config, but current versions use `.env` for common settings.

## Install And Run

```powershell
Copy-Item .env.example .env
python -m pip install -e .
python -m app.main
```

Set these values in `.env` before exposing the gateway to real clients:

```dotenv
GATEWAY_TOKEN_HOST=replace-with-a-strong-token
GATEWAY_TOKEN_ROLE_DEFAULT=replace-with-a-strong-role-token
ARTIFACT_SIGNING_SECRET=replace-with-a-strong-signing-secret
```

Health endpoints:

```powershell
Invoke-RestMethod http://127.0.0.1:8787/healthz
Invoke-RestMethod http://127.0.0.1:8787/readyz
```

## Docker Compose

```powershell
Copy-Item .env.example .env
# Edit .env.
docker compose up --build
```

Compose reads `.env`, builds from the root `Dockerfile`, mounts generated agent config, mounts ComfyUI workflow config, and stores artifacts under `./artifacts`.

## MCP Client URL

Host-local clients:

```toml
[[mcp.servers]]
name = "home"
transport = "sse"
url = "http://127.0.0.1:8787/mcp"
deferred_loading = true
```

Docker clients on the same Compose network:

```toml
[[mcp.servers]]
name = "home"
transport = "sse"
url = "http://home-mcp:8787/mcp"
deferred_loading = true
```

## Agent Setup

1. Edit `.env`:

```dotenv
ENABLED_AGENTS=agent1,agent2
MATRIX_MODULE_ENABLED=true
```

2. Run:

```powershell
.\apply_agent.bat
```

3. Edit each generated `.env.agent.<name>` and fill the Matrix token if that agent sends Matrix messages.

The helper manages these generated files:

- `config/agent/config.agent.<name>.yaml`
- `.env.agent.<name>`

Removing an agent from `ENABLED_AGENTS` and rerunning the helper removes only files that contain the script-managed marker.

## Environment Variables

### Required Secrets

| Variable | Meaning |
| --- | --- |
| `GATEWAY_TOKEN_HOST` | Bearer token for the admin host caller. |
| `GATEWAY_TOKEN_ROLE_DEFAULT` | Bearer token for the default role-play caller. |
| `ARTIFACT_SIGNING_SECRET` | Secret used to sign short-lived artifact download URLs. |

### Server And Artifact Storage

| Variable | Default intent |
| --- | --- |
| `SERVER_HOST` | Bind address. Use `127.0.0.1` for local-only or `0.0.0.0` for Docker/LAN. |
| `SERVER_PORT` | HTTP port, default `8787`. |
| `ARTIFACT_ROOT` | Directory for generated artifacts. |
| `ARTIFACT_PUBLIC_BASE_URL` | Fallback base URL for artifact downloads when request-derived URLs are unavailable. |
| `ARTIFACT_SIGNED_URL_TTL_SECONDS` | Signed artifact URL lifetime. |
| `MAX_ARTIFACT_BYTES` | Maximum generated/uploaded artifact size. |
| `MAX_INLINE_ARTIFACT_BYTES` | Maximum bytes returned by inline image compatibility helper. |

| --- | --- |

### Module Switches

Leave these empty to use `config/config.main.yaml`; set `true` or `false` in `.env` for an explicit runtime override.

| Variable | Module |
| --- | --- |
| `IMAGE_MODULE_ENABLED` | OpenAI-compatible image generation/editing. |
| `LOCAL_IMAGE_MODULE_ENABLED` | Local ComfyUI image generation. |
| `TTS_MODULE_ENABLED` | Text-to-speech. |
| `MATRIX_MODULE_ENABLED` | Matrix send tools. |
| `PRINTER_MODULE_ENABLED` | Printer bridge. |

## Advanced YAML Override

`CONFIG_PATH` still exists for tests and advanced development. If set, the loader merges that YAML over `config/config.main.yaml`. This is not the recommended normal user path.
